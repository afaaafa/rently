# Histórias de Usuário

## Visitante

- Como visitante, quero me cadastrar informando RG, CPF, nome, endereço e profissão, para ter acesso ao sistema.
- Como usuário cadastrado, quero fazer login com minhas credenciais, para acessar minhas funcionalidades.

---

## Cliente

- Como cliente, quero criar um pedido de aluguel selecionando um automóvel disponível, para solicitar a locação.
- Como cliente, quero visualizar o status dos meus pedidos, para saber se estão pendentes, aprovados ou recusados.
- Como cliente, quero editar um pedido ainda não analisado, para corrigir informações antes da avaliação financeira.
- Como cliente, quero cancelar um pedido, para desistir da locação antes da execução do contrato.

---

## Agente da Locadora

- Como agente, quero visualizar pedidos submetidos pelos clientes, para selecioná-los para análise financeira.
- Como agente, quero avaliar um pedido e registrar um parecer (positivo ou negativo), para determinar se o contrato será executado.
- Como agente, quero editar informações de um pedido durante a análise, para corrigir dados antes da aprovação.
- Como agente, quero registrar a propriedade do automóvel alugado (cliente, empresa ou banco) ao executar o contrato, para refletir o tipo de contrato firmado.

---

## Banco

- Como banco, quero associar um contrato de crédito a um pedido aprovado, para financiar a locação do automóvel pelo cliente.
